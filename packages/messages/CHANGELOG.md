# @toad-contracts/messages

## 0.1.0

### Minor Changes

- 13703fe: Add `@toad-contracts/messages`: a minimal Standard Schema superset for message contracts. Provides
  the `MessageTypeCarrier` introspection interface (the message-side analogue of core's
  `ObjectKeysCarrier`), the slim `MessageContract` type, `defineMessageContract`, and the
  `InferConsumerMessage` / `InferPublisherMessage` inference helpers.
